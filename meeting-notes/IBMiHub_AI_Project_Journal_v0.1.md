# Development Approach

We will follow Spec-Driven Development (SDD).

Each feature will have:

-   spec.md
-   plan.md
-   tasks.md
-   test-cases.md

Example:

/specs

000-product-vision

001-authentication

002-dashboard

003-learning

004-ai-tutor

005-5250-lab

006-rpgle-code-explainer

007-subscriptions
